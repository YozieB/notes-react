# Notes React

#### ✨Пет-проект✨

Приложение для заметок

## Использованные технологии и сервисы

- JavaScript
- SCSS
- React.js
- CSS-Modules
- Redux
- Framer Motion
- modern-normalize

## Интересности

- Возможность создавать папки для заметок
- Возможность переименовать заметку прямо на странице
- Удаление заметок/папок
- Анимации
